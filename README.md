# CRDU-Practice-Contacts-cards
Projekt zur Übung von CRUD-Funktionen mit HTML, CSS und JavaScript. Verwendet wird die JSONPlaceholder API, um Daten wie Name, E-Mail, Telefon und Firma zu erstellen, zu laden, zu aktualisieren und zu löschen. Ziel ist das praktische Training von API-Integration und dynamischer Datenverwaltung
